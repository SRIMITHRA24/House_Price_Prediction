# House Price Prediction

## Project Description
This project predicts house prices based on features like area, location, BHK, bathroom count, and furnishing status. Using regression techniques, it provides accurate price estimates to help buyers and sellers make informed decisions.

## Tech Stack
- **Python**
- **Pandas** and **NumPy** for data handling
- **Scikit-Learn** for machine learning models
- **Matplotlib** and **Seaborn** for data visualization
- **Jupyter Notebook** for development

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Model building using regression techniques
- Evaluation metrics: Mean Absolute Error (MAE) and RÂ² Score
- Visualizations to understand trends and correlations
